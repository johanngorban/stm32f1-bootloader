# Терминология

| Термин | Определение |
| --- | --- |
| Слот | Область памяти фиксированного размера для хранения одного бинарного файла |
| Образ | Бинарный файл, расположенный в одном слоте с самого его начала. Имеет размер равный или меньше самого слота. Включает в себя заголовок и тело |
| Заголовок образа (метаданные) | Структура с фиксированными размером (32 байта), полями и их порядком, находящаяся в самом начале образа |
| Тело образа | Часть бинарного файла, кроме заголовка образа. Включает в себя код, константы, переменные и пр. части прошивки |
| Прошивка | Бинарный файл со всей логикой, компилируемый и разрабатываемый отдельно |
| Валидность | Состояние образа, при котором:<br>- CRC в заголовке совпадает с фактическим CRC тела образа<br>- Размер тела образа больше 0<br>- Размер образа не больше размера слота<br>- Магическое число из заголовка совпадает с ожидаемым загрузчиком |
| Верификация | Процесс, при котором проверяется, валиден ли образ |
