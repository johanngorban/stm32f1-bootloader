# CLI синтаксис утилиты bootctl

Каждая команда, которая выводит что-либо, поддерживает флаг `--json`, отображающий вывод в формате JSON.

## Описание флагов

- `--no-verify`. Пропускает верификацию прошивки. Бинарник просто загружается и отдается статус "успешно".
- `--json`. Выводит в формате JSON. Если не установлен, выводит в человекочитаемом виде.
- `--slot <1|2>`. Указывает слот. Допустимые значения 1 или 2. По умолчанию задан первый слот.

## Помощь

```
bootctl help
```

### Вывод

```
Syntax:

bootctl command [flags]

Commands:

 - flash [--slot <1|2>] [--no-verify] [--json]
 - status [--slot <1|2>] [--json]
 - run [--slot <1|2>] [--json]
```

## Загрузка прошивки

По умолчанию:

- Загрузка происходит в первый слот
- Происходит верификация загруженной прошивки
- Вывод происходит в человекочитаемом виде (не JSON)

```
bootctl flash [--slot <1|2>] [--no-verify] [--json]
```

## Вывод информации о прошивках

По умолчанию:

- Выводится информация об обоих образах (версия, CRC)
- Вывод происходит в человекочитаемом виде (не JSON)

```
bootctl status [--slot <1|2>] [--json]
```

## Запуск прошивки

По умолчанию:

- Запускается первый слот
- Вывод происходит в человекочитаемом виде (не JSON)

```
bootctl run [--slot <1|2>] [--json]
```
